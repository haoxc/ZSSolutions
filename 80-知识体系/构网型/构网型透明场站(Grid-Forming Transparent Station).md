# 构网型透明场站(Grid-Forming Transparent Station)

## 核心定义

> **构网型透明场站**是指新能源场站（风光储）具备**构网型（Grid-Forming, GFM）**逆变器特性——主动为电网提供惯量支撑、频率调节、电压调节等系统级服务——同时将涉网性能数据、发电运行数据**实时透明化展示**给电网调度端和场站运维端的新一代运营模式。

它回答了两个问题：
- **构网型**：场站从"跟着电网走"变为"参与建网"（Grid-Following → Grid-Forming）
- **透明场站**：场站的电网交互行为不再是黑箱，而是可观测、可评估、可审计

## 最小心智模型

```
传统场站 (Grid-Following):
  电网 ──指令──▶ 场站逆变器 ──被动跟随──▶ 输出功率

构网型场站 (Grid-Forming):
  电网 ◀──惯量/调频/调压── 场站逆变器 ──主动支撑──▶ 输出功率
                                     │
                                     ▼
                              透明展示层 (4-5页面)
                              涉网性能 | 发电性能
```

**核心转变**：逆变器从"电流源"变为"电压源"——自主建立电压频率参考，而非锁相跟随电网。

## 构网型 vs 跟网型

| 维度 | 跟网型 (GFL) | 构网型 (GFM) |
|------|-------------|-------------|
| 控制原理 | 锁相环(PLL)跟随电网电压 | 自主建立电压频率参考(V/f) |
| 惯量响应 | 无天然惯量 | 虚拟惯量，响应 < 100ms |
| 一次调频 | 需AGC指令间接参与 | 自主响应频率偏差，下垂控制 |
| 电压支撑 | 无功功率受限于PLL | 自主调压，无功限幅内优先 |
| 弱网适应性 | 弱网易失稳 (SCR < 1.5) | 天然适配弱网 (SCR可低至1.0) |
| 孤岛运行 | 不可 | 可（支持黑启动） |
| 过流能力 | 1.1pu 速切 | 1.2~1.3pu 暂态支撑 |

**关键认知**：构网型并不是"更智能"的控制，而是**更基础的物理层控制范式切换**。它位于逆变器控制的最内环（电压电流环），而非外环的优化调度。

## 透明场站展示内容

在 JBFG 项目中，客户要求的场景范围（4~5页面）可归纳为两类展示：

### 涉网性能展示

| 指标 | 说明 | 电网要求依据 |
|------|------|------------|
| 惯量响应 | 频率变化率(RoCoF)抑制能力，响应时间 & 贡献能量 | GB/T 40595-2021 |
| 一次调频 | 频率偏差→有功调节，响应时间 & 调节精度 & 死区 | GB/T 19963.1-2021 |
| AGC 调节性能 | 调节速率、调节精度、响应时间（K1/K2/K3 指标） | 《两个细则》 |
| AVC 调节性能 | 电压无功调节，功率因数范围、响应时间 | GB/T 19963.1-2021 |
| 高低电压穿越 | LVRT/HVRT 期间逆变器表现记录 | GB/T 36995-2018 |
| 有功功率控制 | 限功率指令跟踪精度、爬坡速率 | 电网调度协议 |

### 发电性能展示

| 指标 | 说明 |
|------|------|
| 实时发电功率 | 总有功/无功、预测 vs 实际对比 |
| 弃风/弃光率 | 因电网限电导致的发电损失统计 |
| 综合效率 (PR) | 理论发电量 vs 实际发电量之比 |
| 厂用电率 | 站内自耗电占比 |
| 等效利用小时数 | 反映资产利用率 |

## 在 JBFG 项目中的定位

```
客户需求原文:
  "构网型透明场站信息展示，涉网性能、发电性能 4~5个页面"
```

**评估**：
- 客户将此作为一个**展示需求**提出，其背后隐含的技术前提（场站逆变器是否已具备构网型能力）被省略了
- 如果客户场站已完成构网型改造/新建，那这 4~5 页面就是纯粹的**数据可视化层**开发
- 如果客户仅想做"透明"而不要求"构网"（即仅展示性能指标，不涉及控制层），那就退化为**增强型 SCADA 可视化**

**方案书关键假设**：需要在方案书中明确以下假设——
> 本方案仅覆盖构网型场站的**可视化展示层**，构网型逆变器改造/新建由场站电气侧独立完成。

若客户期望方案同时覆盖逆变器层改造，项目范围将横跨电力电子控制 + 数字化可视化两个完全不同的技术域，报价和工期需重新评估。
