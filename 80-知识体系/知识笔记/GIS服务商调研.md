# GIS 服务厂商调研

> 调研目的：为 JBFG 方案书中的 GIS/三维可视化模块提供定价基准和竞争对标。

**定位**：国产 GIS 龙头，市占率第一，信创首选。

| 产品                         | 价格             | 价格类型                   |
| -------------------------- | -------------- | ---------------------- |

| 跨平台二三维桌面GIS V11            | **¥107,150/套** | 福建采购（2025，2套共¥349,960） |
| SuperMap GIS企业级基础平台        | **¥635,456/次** | 华为云商城标价                |
| SuperMap Online            | **¥100/月**     | 云订阅                    |

> **核心能力**：成熟的二三维一体化、WebGL 无插件访问、支持海量空间数据、信创适配（麒麟/统信/达梦/人大金仓）。

## 四、JBFG 方案书中的定价策略建议

### 定价对标

| 报价模块                           | 对标参考              | 建议水位          |
| ------------------------------ | ----------------- | ------------- |
| 三维可视化门户（Three.js + Cesium 双引擎） | 华电 233.5 万（纯软件开发） | ¥200~300万     |
| 数据网关（多源数据接入适配）                 | 无直接对标（或从华电项目拆分）   | ¥50~80万       |
| 三维建模（风/光/储模型）                  | 华电项目中隐含在平台费内      | ¥50~100万      |
| 实施/集成/培训                       | 华电项目内包含           | ¥50~80万       |
| **「报价总量级参考」**                  |                   | **¥350~560万** |

> ⚠️ 上述报价为**纯软件/平台开发**水位线，不含任何 AI 诊断模块。如客户要求 AI 能力，需另加约 ¥150~300 万（参考陆上风电智慧运维方案的 AI 模块估算）。

### 竞品差异化策略

与上述竞品相比，JBFG 方案书的差异化卖点：

| 竞品短板                        | 我们可打的点                           |
| --------------------------- | -------------------------------- |
| 超图/中地：通用 GIS 平台，新能源行业需要深度定制 | **行业理解**：我们对风光储的业务场景更熟悉，无需客户解释需求 |
| 恒华科技：偏设计阶段，运维可视化能力有限        | **运维视角**：方案侧重**生产运行**而非工程设计      |
| 国遥新天地/广泽大地：项目价高（千万级），灵活度低   | **性价比**：提供 300~500万级别的**轻量化方案**  |
| 三峡智控：央企自研，不对外输出             | **可复制性**：本方案可产品化，支持多场站部署         |

## 五、与 JBFG 项目的映射

| GIS 能力需求点 | 自研方案 | 采购超图/中地方案 | 结论 |
|--------------|---------|----------------|------|
| 地理坐标显示（WGS84投影） | Cesium 原生支持 | ✅ | 自研可覆盖 |
| 场区三维地形重建 | Three.js + DEM数据 | ✅ | 自研可覆盖 |
| 空间分析（缓冲区/叠置分析） | 需自研 | ✅ 平台自带 | **采购更优**（复杂度高） |
| 坐标转换与配准 | 需自研 | ✅ 平台自带 | **采购更优** |
| 可视化渲染 | Three.js 灵活可控 | ⚠️ 超图渲染能力有限 | **自研更优** |

**建议**：核心可视化展示自研（Three.js + Cesium），空间分析功能可集成超图 iServer 做后端支撑（采购 ¥60万级服务器许可），形成"自研前端 + 采购后端"的混合路线。
