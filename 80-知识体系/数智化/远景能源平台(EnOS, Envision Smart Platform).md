# 远景能源平台 (EnOS, Envision Smart Platform)

> 远景科技集团旗下四大业务：远景能源（风机）、远景动力（电池）、远景智能（EnOS 平台）、远景资本。风机订单 2022-2024 连续三年全球第一。

## EnOS™ 智能物联平台

远景智能（Univers）开发的智能物联网操作系统/技术底座。

### 三大核心模块

| 模块 | 能力 |
|------|------|
| **设备连接与管理** | 覆盖 90%+ 主流规约协议，支持 MQTT/CoAP/HTTP，异构设备高效安全接入 |
| **数据管理与分析** | 跨时序库/关系库/图数据库的端到端数据治理 |
| **应用构建** | 低代码/无代码 + 高阶 API/SDK |

### 部署方式
- **边缘端**：EnOS Edge 智能边缘网关
- **云端**：EnOS Cloud
