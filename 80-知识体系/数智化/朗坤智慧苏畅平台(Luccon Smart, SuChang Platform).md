# 朗坤智慧苏畅平台 (Luccon Smart, SuChang Platform)

> 成立于 1999 年，总部南京鼓楼。核心平台"苏畅"为国家级双跨工业互联网平台（2023 年全国 A 级第 10），近期获评工信部"制造业单项冠军"。

## 核心平台

### 苏畅工业互联网平台
- 国家级"跨行业跨领域"（双跨）平台
- 与 28 家信创头部企业完成六大领域技术适配
- 构建"工业智能平台 + 大模型 + 工业 AI 应用"协同体系

### 下一代能源集团智慧化生产管控平台

面向大型能源集团，覆盖 **80+ 智慧化应用、1500+ 数智模型**：

| 领域 | 关键能力 |
|------|----------|
| **运行生产** | 智慧监盘、运行优化、预警推荐、智能交接班 |
| **设备维护** | AI+机理融合状态评估、故障定位、预测性维护 |
| **安全管理** | 智能两票、违章抓拍、风险预警、安全一张图 |
| **经营管理** | 实时成本计算、智能化煤场、管理驾驶舱 |

### 苏畅瑶光工业大模型

- **大模型 + 小模型** 协同架构
- 小模型：精确定位设备故障（"专科医生"）
- 大模型：调用知识库智能推荐处置策略（"智能导医台"）

## 行业典型案例

### 国电电力"云边协同"（标杆）
- 平台技术鉴定为 **"国际领先"**
- 接入 **12 家海内外电厂、37 台机组**
- 综合煤耗下降 **0.45g/kWh**
- 机组非停次数同比下降 **38%**
- 累计预警 **55 万+ 条**，准确率 **>90%**
- "五升一降"：设备可靠性、运行水平、安全管控、专业技术、成本管控提升，劳动强度降低

### 国家电投贵州织金智慧电厂
- 单台机组部署 **1300+** 机理模型 + AI 算法模型
- 年节约燃料和物料成本 **400 万元+**
- 实现数据可见 → 态势可感、决策可用

### 跨行业（钢铁）

| 项目 | 成效 |
|------|------|
| 太原钢铁 AI 智慧能耗优化 | 单台除尘风机能耗降 **24.8%**，年减碳 **1286 吨** |
| 梅山钢铁产线能效优化 | 40 台风机上线，全厂 100 台年省电费 **~3000 万元** |

### 煤炭
- 第七届工业互联网大赛全国总决赛 **冠军**
- 煤机设备预警准确率 **95%**
- 覆盖 10 类核心设备，28 智能体协同

## 核心能力亮点

| 能力 | 说明 |
|------|------|
| **预测性维护** | "定期修"→"状态修、预测修"，预警准确率 >95% |
| **智慧监盘** | AI 仪表盘→操盘手，释放 **30%+** 运维人力 |
| **工业 AI 智能体** | 覆盖设备诊断、生产管理、企业经营 |
| **低代码建模** | 拖拉拽完或工业机理 + AI 融合建模 |
| **云边协同** | 云端全局优化，边缘毫秒级实时响应 |

## 战略方向

- **"AI All-In"** 战略
- 大型能源集团：全链条穿透式智能管控
- 中小企业："小快轻准"SaaS 化应用
- 重点行业：发电 → 钢铁 → 化工 → 煤矿
