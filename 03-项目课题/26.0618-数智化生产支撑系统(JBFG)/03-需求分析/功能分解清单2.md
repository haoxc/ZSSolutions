# 功能分解清单2（操作级）

> **颗粒度定义**：操作级 = 每个子功能拆解为具体的用户/系统操作动作（查看/点击/配置/查询/导出/创建/编辑/删除/上传/下载/告警/分析/执行/审批等）。
> **交叉依据**：技术规范书 §5.2（功能要求）+ §5.3（性能要求）+ §5.4（接口要求）+ 需求澄清问卷 Q1-Q11。
> **交付范围**：标注「不在交付范围」的功能域/模块 = 客户已采购南瑞运维平台或独立子系统，我方仅做接口对接 + 数据接收展示。

## 操作类型速查

| 操作类型   | 含义         | 示例            |
| ------ | ---------- | ------------- |

| 🔎 查询  | 条件检索历史数据   | 按时间段查询告警记录    |
| 📥 导出  | 数据/报表导出下载  | 导出运维报告为 PDF   |
| 📤 上传  | 文件/模型/图片上传 | 上传三维模型 GLB 文件 |
