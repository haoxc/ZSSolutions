# 出版级排版规范说明书

> 适用模板：模板 A（完整应标版）/ 模板 B（技术方案版）
> 目标品质：出版级——符合 GB/T 9704-2012 公文格式规范 + 出版印刷行业通行标准

---

## 一、总则

### 1.1 设计原则

| 原则 | 说明 |
|------|------|
| **统一** | 全文档不超过 2 种中文字体、2 种英文字体，保持视觉一致性 |
| **层级** | 标题层级显式区分（字号递进），同一层级在全文中样式绝对一致 |
| **呼吸** | 留白优先于填满——段落间距、页边距、图文间距均有明确值 |
| **可追溯** | 每个排版决策对应一个可检验的约束条件（字号/行距/色值） |

### 1.2 适用标准

| 标准 | 适用范围 |
|------|---------|
| GB/T 9704-2012 | 页面设置、装订边距、页码位置参考 |
| GB/T 1.1-2020 | 章节层级编号规范参考 |
| GB/T 7714-2015 | 参考文献著录规则（如有引用） |
| CY/T 35-2001 | 出版物排版质量要求（出版级参考） |

## 二、字体系统

### 2.1 字体矩阵

| 层级 | 中文字体 | 英文字体 | 字重 | 用途 |
|------|---------|---------|:----:|------|
| 封面大标题 | 思源宋体 Heavy | TNR Bold | Heavy/Bold | 方案名称，印象锚点 |
| H1 | 黑体 (SimHei) | TNR Bold | Bold | 章标题 |
| H2 | 黑体 (SimHei) | TNR Bold | Bold | 节标题 |
| H3 | 黑体 (SimHei) | TNR Bold | Bold | 小节标题 |
| 正文 | 宋体 (SimSun) | TNR | Regular | 全部正文内容 |
| 图表标题 | 宋体 (SimSun) | TNR | Bold | 图名/表名 |
| 表格内容 | 宋体 (SimSun) | TNR | Regular | 表格数据 |
| 代码/参数 | 宋体 (SimSun) | Consolas | Regular | 等宽场景 |
| 页眉页脚 | 宋体 (SimSun) | TNR | Regular | 页眉/页码 |

### 2.2 字体回退策略（重要）

> **问题**：思源字体（Source Han Serif/Sans）在部分评审方电脑上可能未安装。
> **策略**：回退到系统自带字体，避免渲染异常。

| 首选字体 | 第一次回退 | 第二次回退 | Word 设置方法 |
|---------|-----------|-----------|-------------|
| 思源宋体 Heavy | 宋体 (SimSun) | — | 嵌入式字体（嵌入仅所用字符） |
| 思源黑体 | 黑体 (SimHei) | — | 同上 |
| Consolas | Courier New | — | 常规回退 |

**操作步骤：**
1. 文件 → 选项 → 保存
2. 勾选「将字体嵌入文件」
3. 选择「仅嵌入文档中使用的字符」（减小文件体积）
4. 注意：部分评审方可能要求不嵌入字体以控制文件大小，此时需确认评审方环境已安装所选字体

### 2.3 字号规范（单位：pt）

```
封面标题: 28     H1: 18     H2: 14     H3: 13     H4: 12
正文: 12        图表名: 10.5    表格: 10.5    页眉: 9    页码: 9
```

### 2.4 中英文混排规则

| 规则 | 示例 |
|------|------|
| 中英文之间加 1/4 空格（或手动空格） | "USDF 底座" 而非 "USDF底座" |
| 中文与数字之间加空格 | "100ms 以内" 而非 "100ms以内" |
| 英文与数字之间不加空格（标准习惯除外） | "V1.0" 而非 "V 1.0" |
| 全角标点与半角混排 | 中文段落用全角标点，英文/数字用半角 |
| 引号嵌套 | 外双内单：他说：「报告中提到'空间对象操作系统'这个概念。」|

## 三、段落系统

### 3.1 行距规范

| 样式 | 行距设置 | 实际值 |
|------|---------|--------|
| 正文 | 固定值 22pt | 约 1.3 倍（12pt 字号时） |
| H1-H4 | 1.5 倍行距 | — |
| 表格内容 | 单倍行距 | — |
| 图表标题 | 单倍行距 | — |
| 页眉页脚 | 单倍行距 | — |

### 3.2 段落间距

| 样式 | 段前 | 段后 |
|------|:---:|:---:|
| H1 | 18pt | 12pt |
| H2 | 12pt | 6pt |
| H3 | 6pt | 6pt |
| H4 | 6pt | 3pt |
| 正文 | 0 | 0 |
| 图名 | 3pt | 6pt |
| 表名 | 6pt | 3pt |
| 列表项 | 0 | 3pt |

### 3.3 孤行控制（重要）

**设置方法：**
1. 选中所有正文段落
2. 段落 → 换行和分页
3. 勾选「孤行控制」
4. 勾选「与下段同页」（标题专用）
5. 勾选「段中不分页」（表格、图表前后段落）

**出版级标准：**
- 禁止标题位于页面最后一行（孤行）
- 禁止标题后无正文直接翻页
- 禁止表格标题与表格主体分离
- 禁止图表跨页不标注「续表」

## 四、页面系统

### 4.1 页面设置

| 参数 | 值 | 说明 |
|------|-----|------|
| 纸张 | A4 (210×297mm) | 纵向 |
| 上边距 | 2.54cm (1 inch) | 含页眉 1.5cm |
| 下边距 | 2.54cm | 含页脚 1.5cm |
| 左边距 | 3.17cm (1.25 inch) | 装订预留，双面打印时左右对称 |
| 右边距 | 3.17cm | — |
| 装订线 | 0cm | 不在页边距外加装订线 |
| 页眉距边界 | 1.5cm | — |
| 页脚距边界 | 1.5cm | — |

### 4.2 分节规范

```
节 1: 封面 (无页码, 首页不同)
节 2: 版权声明+修订记录 (罗马数字 Ⅰ, Ⅱ)
节 3: 目录+图表索引 (罗马数字 Ⅲ, Ⅳ, Ⅴ)
节 4: 正文 (阿拉伯数字 1, 2, 3…, 从1开始)
节 5: 附录 (续阿拉伯数字)
```

**分节操作：**
1. 在封面的最后一页末尾插入「布局 → 分隔符 → 下一页」
2. 进入页眉页脚编辑状态
3. 对节 2，关闭「链接到前一节」
4. 设置页码格式：节 2-3 为罗马数字，节 4-5 为阿拉伯数字
5. 节 4 页码起始设为 1

### 4.3 页码格式

| 位置 | 设计 |
|------|------|
| 前置部分 | 罗马数字 Ⅰ、Ⅱ、Ⅲ…，页脚居中对齐 |
| 正文 | 阿拉伯数字，格式 `- 1 -`，页脚居中对齐 |
| 附录 | 续接正文页码 |
| 封面 | 无页码 |

### 4.4 页眉设计

**正文页（节 4+）：**

模板 A：
- 左侧：公司名称（9pt 宋体，灰色 #666666）
- 右侧：章标题（自动域，9pt 宋体，灰色）
- 底部 0.5pt 浅灰分隔线（颜色 #CCCCCC）

模板 B：
- 左侧：方案名称简写（9pt 宋体，灰色）
- 右侧：章标题
- 底部 0.5pt 浅灰分隔线

**章首页：** 设置「首页不同」，章首页不显示页眉。

## 五、图表系统

### 5.1 表格规范

| 元素 | 规范 |
|------|------|
| 表名位置 | 表上方居中 |
| 表名字体 | 宋体 10.5pt 加粗，黑色 |
| 表头字体 | 宋体 10.5pt 加粗，白字 |
| 表头底色 | 模板 A：深蓝 #1B3A5C / 模板 B：深灰 #4D4D4D |
| 表身字体 | 宋体 10.5pt，灰色 #333333 |
| 表身行距 | 单倍行距 |
| 外边框 | 1.5pt，深蓝/深灰 |
| 内边框 | 0.5pt，浅灰 #CCCCCC |
| 对齐 | 文本居左/数字居右 |
| 跨页 | 标题行重复 |
| 表注 | 表下方，宋体 9pt，灰色 #666666 |

### 5.2 插图规范

| 元素 | 规范 |
|------|------|
| 图名位置 | 图下方居中 |
| 图名字体 | 宋体 10.5pt 加粗 |
| 图片质量 | ≥ 300dpi（印刷质量） |
| 图片格式 | 矢量图优先（.emf / .wmf），其次高分辨率 .png |
| 架构图 | 统一线宽（2pt 主框线 / 1pt 辅助线），统一配色（符合本规范色板） |
| 截图类 | 标注数据来源或测试环境，避免误解为产品截图 |
| 图片对齐 | 居中，不可「嵌入式」环绕（用「上下型」环绕） |

### 5.3 编号规范

```
图编号: 图 1-1, 图 1-2, 图 2-1...
表编号: 表 1-1, 表 2-1, 表 2-2...
公式: (1-1), (1-2)...
```

**操作：**
- 使用 Word「引用 → 插入题注」功能
- 选择「自动插入题注」可自动化

## 六、化学/工程表达式与特殊字符

### 6.1 数字与单位

| 规则 | 正确 | 错误 |
|------|------|------|
| 数字与单位间加空格 | 100 ms | 100ms（技术规格中可接受） |
| 单位使用国际符号 | ≤100 ms | <=100 ms 或 小于等于100毫秒 |
| 使用乘号 × 而非 x | 20 × 30 | 20 x 30 |
| 温度符号 | °C (U+00B0 + C) | °C（独立字符） |

### 6.2 工程参数

| 规则 | 示例 |
|------|------|
| 测量精度 | ±0.5% |
| 范围表示 | 100–200（EN dash, U+2013） |
| 百分号 | %, 数字后不空格（中文中） |
| 比值 | 1:2 或 1∶2 |

## 七、打印与输出规范

### 7.1 纸张与色彩

| 场景 | 纸张 | 色彩 |
|------|------|------|
| 正式投标（打印版） | 80g 以上胶版纸/哑粉纸 | CMYK 印刷 |
| 电子版 | — | sRGB（屏幕显示） |
| 技术评审（内部） | 70g 复印纸 | 灰度（B&W friendly） |

### 7.2 输出检查清单

**每次输出前逐项检查：**

- [ ] 所有标题层级正确（无跳过 H1→H3 的情况）
- [ ] 目录域已更新（Ctrl+A → F9）
- [ ] 所有交叉引用已更新（Ctrl+A → F9）
- [ ] 图表编号连续（无重复编号）
- [ ] 表格未意外跨页断开
- [ ] 封面无页码
- [ ] 正文页码从 1 开始
- [ ] 无孤行（标题在页末、段末只有一行）
- [ ] 字体已嵌入（或已确认评审方环境兼容）
- [ ] 转 PDF 后检查：字体渲染、图表位置、页码一致性

### 7.3 PDF 输出设置

```
文件 → 导出 → 创建 PDF/XPS
选项：
  - 发布为 PDF/A 兼容 (长期存档)
  - 优化图像质量 (高分辨率)
  - 不勾选「轻最小化大小」
  - 勾选「文档结构标签」(无障碍)
```

## 八、常见问题与修复

| 问题 | 原因 | 修复 |
|------|------|------|
| 图表编号「图 1-1」变为「图 1?1」 | 域代码未更新 | Ctrl+A → F9 更新所有域 |
| 标题跑到页面最后一行 | 未设置孤行控制 | 段落 → 换行和分页 → 孤行控制 |
| 表格标题和表身跨页分离 | 未勾选「标题行重复」 | 表格工具 → 布局 → 重复标题行 |
| 封面不小心被编页码 | 未使用分节符 | 插入分节符，封面节取消「链接到前一节」|
| 中英文字体不一致 | 字体未同时设置中文和英文字体 | 修改样式时同时设置中/西文字体 |
| 图片打印模糊 | 屏幕截图分辨率不足 | 替换为 ≥300dpi 原图 |
| 行距在不同电脑上显示不同 | 使用了「多倍行距」而非「固定值」 | 统一使用「固定值」行距 |

## 九、一键排版速查卡

> 将此页打印出来贴在工位上。

### 字体设置
```
文中  → 宋体 12pt | 英数 → TNR 12pt | 行距 → 固定 22pt
H1   → 黑体 18pt | 段前 18 段后 12 | 1.5 倍
H2   → 黑体 14pt | 段前 12 段后 6  | 1.5 倍
H3   → 黑体 13pt Bold | 段前 6 段后 6 | 1.5 倍
```

### 页面设置
```
上 2.54 | 下 2.54 | 左 3.17 | 右 3.17 (cm)
封面：首页不同，无页码
前置：罗马数字 ⅠⅡⅢ
正文：阿拉伯数字从 1 开始
```

### 图表
```
表名在上，图名在下 | 宋体 10.5pt Bold
表头深蓝底白字 | 表格边框 1.5pt 深蓝 + 0.5pt 浅灰
```

### 输出前
```
[ ] 更新目录域  Ctrl+A → F9
[ ] 更新交叉引用  Ctrl+A → F9
[ ] 检查孤行
[ ] 检查跨页表格
[ ] 转 PDF 再检查一遍
```
