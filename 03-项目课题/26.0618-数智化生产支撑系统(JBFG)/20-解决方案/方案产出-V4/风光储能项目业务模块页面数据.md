## 1. 页面结构总览

### 1.1 一级导航

#### 导航栏显示规则

- 导航栏固定在页面左侧。
- 导航栏中的各页面入口纵向排列。
- 导航栏入口全部使用文字显示，不使用图标显示。
- 导航栏文字项包括：风机管理、逆变器管理、平滑出力、跟踪计划、主接线路、温度云图、路线展示、储能系统。
- 当前选中页面需要有明确的文字高亮状态。

| 一级分组 | 页面/模块 | 页面定位 |
|---|---|---|
| 全局总览 | 资产概览（默认页面） | 项目资产地图与场站分布总览，是全局总览默认进入页 |
| 全局总览 | 场景概览 | 资产概览下的子页面，展示源网荷储业务关系与核心指标 |
| 全局总览 | 二维组态 | 资产概览下的子页面，展示源网荷储二维拓扑/组态关系 |
| 导航栏 | 风机管理 | 导航栏进入的主页面，带三维场景、信息面板和 PoI 交互 |
| 风机管理子页面 | 风机管理面板 | 从风机管理主页面点击按钮进入，只显示数据，不带场景 |
| 导航栏 | 逆变器管理 | 导航栏进入的主页面，带三维场景、信息面板和 PoI 交互 |
| 逆变器管理子页面 | 逆变器管理面板 | 从逆变器管理主页面点击按钮进入，只显示数据，不带场景 |
| 导航栏 | 平滑出力 | 风光储联合出力、波动率、储能调节能力展示 |
| 导航栏 | 跟踪计划 | 计划出力、预测出力、实际出力和偏差跟踪 |
| 导航栏 | 主接线路 | 变电站一次接线图展示 |
| 导航栏 | 温度云图 | 储能系统温度分布可视化 |
| 导航栏 | 路线展示 | 储能系统、调度中心、变电站线路布局展示 |
| 导航栏 | 储能系统 | 导航栏进入的主页面，带三维场景、信息面板和 PoI 交互 |
| 储能系统子页面 | 储能单元监控 | 从储能系统主页面点击按钮进入，只显示数据，不带场景 |
| 储能系统子页面 | 储能单元控制 | 从储能系统主页面点击按钮进入，只显示数据，不带场景 |
| 储能系统子页面 | PCS 变流器 | 从储能系统主页面点击按钮进入，只显示数据，不带场景 |

### 1.2 通用页面组成

每个业务页面建议按以下结构复刻：

| 区域 | 内容说明 |
|---|---|
| 导航栏内容 | 左侧纵向文字导航、当前页面入口、页面名称、可切换子页面或功能模式 |
| 场景内容 | 三维/二维场景中的模型、空间关系、线路、动画和可视化效果 |
| 信息面板内容 | 页面默认展示的统计指标、趋势图、状态列表和业务卡片 |
| PoI 内容 | 场景中可点击的点位对象，例如风机、光伏方阵、储能集装箱、PCS |
| PoI 详情页内容 | 点击 PoI 后出现的弹窗、抽屉或详情页，包含运行数据、告警、趋势和控制 |
| 操作能力 | 下钻、切换设备、远程控制、查看事件、查看可视化层等交互 |

## 3. 风机管理

### 3.1 导航栏内容

- 一级入口：左侧纵向文字导航
- 页面名称：风机管理
- 页面定位：风电场三维运行、风机状态监控、风机详情分析和远程控制
- 页面关系：从左侧导航点击“风机管理”文字项进入该主页面；该页面带三维场景、信息面板和 PoI 交互；点击“风机管理面板”按钮进入纯数据子页面

### 3.2 场景内容

| 场景元素 | 展示内容 | 复刻要点 |
|---|---|---|
| 风电场三维场景 | 陆上风机、风机编号、光伏方阵、光伏方阵编号 | 风机模型需要具备编号标识和状态映射 |
| 风机空间分布 | 各风机在场站内的位置 | 支持从场景点选风机 |
| 光伏方阵辅助展示 | 光伏方阵及编号 | 可作为同场站能源资产补充展示 |

### 3.3 信息面板内容

| 面板 | 数据字段/图表 |
|---|---|
| 风机概览 | 装机台数、装机容量、运行数量、故障数量、维护数量、待机数量、未知数量、离线数量、风机矩阵状态 |

### 3.4 PoI 内容

| 风机 | 实时运行数据 |

### 3.6 子页面：风机管理面板

- 进入方式：在带场景的风机管理主页面中点击“风机管理面板”按钮进入
- 页面类型：纯数据子页面
- 场景要求：不显示三维场景
- 内容定位：集中展示单台或多台风机的数据、趋势、状态记录和远程控制

| 面板区块 | 数据字段/功能 |
|---|---|
| 风电场信息 | 风机编号、风速、运行状态、功率等矩阵化信息 |
| 主要运行参数 | 风速、功率、环境温度、发电机转速、偏航速度、对风角度、偏航角度、叶片角度 |
| 涉网信息 | A 相电流、B 相电流、C 相电流、AVC、无功功率、功率因数、电网频率 |
| 状态变换记录 | 风机编号、状态、开始时间、结束时间 |
| 设备切换 | F01、F02、F03 等风机编号下拉选择 |
| 实时运行趋势 | 风速、叶轮转速、发电机转速、功率变化曲线 |
| 发电机温度趋势 | 发电机定子 U 温度、发电机定子 V 温度、发电机定子 W 温度变化曲线 |
